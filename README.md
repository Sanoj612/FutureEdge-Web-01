# FutureEdge Frontend

FutureEdge is a modern web application designed to connect job seekers with employers using AI-powered recommendations. The frontend provides an intuitive interface for job seekers to manage profiles, upload resumes, and receive tailored job suggestions, as well as for employers to create and manage job postings efficiently.

## Features
- Job Seeker Dashboard with AI-powered job recommendations
- Resume upload and AI-based resume parsing for skills, experience, and job titles
- Employer Dashboard for creating, updating, and managing job postings
- Real-time job search with filters by location, skills, job type, and experience level
- Interactive UI components: Cards, Badges, Inputs, Buttons, and Alerts
- Integration with OpenAI API for resume parsing and job matching
